Free Graphic Tools

Like any expert in anything, you will need good tools to do good work.  So I have listed a few of the best freeware and shareware utilities on the net.

Graphic Editors - Get great programs to do your own graphics

Paint Shop Pro - This is the shareware graphics editor used by many pros.
GifClean32 - This free utility makes your gif files real small so they download faster.

Animated GIF Editors - Make your own moving pictures.

GIF Construction Set - My preffered animated GIF program.  This one is a little advanced, but well worth learning.
Microsoft Gif Animator - This one is very easy to learn, and does a great job too.
Ulead Gif Animator - Another easy to use program that comes with great animated gif files.  This page has other great graphic utilities too.

Other Tools - There are tools here for everything web pageish.

MAP THIS! - Ever wonder how they made it possible to click on only parts of a graphic?  Well this program makes it easy to do and it's FREE!
HTML Reference Library - This program will show you all you want to know as far as HTML is concerned.
