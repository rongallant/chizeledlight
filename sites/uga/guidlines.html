Guidelines for submitting artwork
  1. You can display as many pieces of artwork as you like on one condition... (see rule #2 for the one condition)

  2. Total file size must be kept under 500KB. That means all your files together are 500KB or less. With the volume of artists we are expecting, we have to limit space until we can afford a larger web space.

  3. You should send the files together in a single ZIP file which includes your BIO (see one of the portfolios for an example). The name of the file should be your full name (.i.e. michael_kennicutt.zip) **WinZip is available here: www.winzip.com This is shareware for creating zip files.

  4. All artwork should be in JPG or GIF format. These are the only viable formats to display artwork on the Internet and we don't have the time to convert hundreds of images from other formats.

  5. There cannot be any legible Internet Addresses or email addresses in submitted artwork.

  6. All artwork should be watermarked. If you do not know how to do this, we can do it for you (but it saves me time if you can do this yourself.)

  7. Send the completed zip file to Tony Mendez threw ICQ (ICQ# 45466770), or email him at .

Watermarking An Image
  • First off, I am not talking about digital watermarking, which is a process where the file itself has hidden information written inside. This type of file protection can be foiled by simply doing a screen shot of the image. In order to protect your artwork, an actual semi-transparent logo of some sort should be overlayed on top of the image itself and saved that way.

  • To do this type of watermarking, you will need a graphics program that does layering. Photo Shop is very good at this, but Paint Shop Pro will do just as well and is much more affordable.

  • In Paint Shop Pro you first open your image and create a second layer with the layer tool. In the layers window, this new layer should be on top of the original image layer. Now use the text tool to put a message of some sort (i.e. United Graphic Artists) with a color opposite of the image (i.e. for dark images, use white text). Duplicate that message in the new layer until it covers the entire image. Now use the slide bar in the layers window to change the transparency from 100% to 10%. You now should have a semi-transparent water mark over your image so no one can just take and reuse that image. Go to any of the artists portfolios to see examples.
